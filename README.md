# Taller3_AccesTreeB-
Simulación de Sistema de Archivos masivo en C++ utilizando una arquitectura híbrida: Grafo Dirigido (Lógica) indexado por un Árbol B+ (Índice Físico). Implementación manual sin librerías de contenedores.
# Compilacion
g++ -std=c++17 -O2 -Wall -Wextra -pedantic main.cpp ArbolBPlus.cpp NodoBPlusBase.cpp NodoBPlusHoja.cpp NodoBPlusInterno.cpp NodoGrafo.cpp NodoDirectorio.cpp NodoArchivo.cpp -o Taller3_AccesTreeB-
# Ejecucion
./Taller3_AccesTreeB-
